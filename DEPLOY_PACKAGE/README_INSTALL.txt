================================================== 
          QUIZ APP INSTALLATION GUIDE 
================================================== 
 
1. Install Node.js 
   - Download and install from: https://nodejs.org/ 
   - This is required to run the server. 
 
2. Start the App 
   - Double-click 'START_APP.bat'. 
   - The first time you run it, it will automatically 
     install necessary files (this takes 1-2 mins). 
 
3. Connect 
   - The app will open and show you the URL to use. 
     (e.g., http://192.168.1.5:3001)
