# Nava Bharath Quiz Application

This is the production-ready version of the Nava Bharath Quiz App.

## Folders
- `/client`: Frontend code (React + Vite)
- `/server`: Backend code (Node.js + Socket.io)

## How to Deploy (Quick Start)
1. **Upload to GitHub**: Upload this entire `deploy_ready` folder to a new GitHub repository.
2. **Frontend (Vercel/Netlify)**: Point to the `client` directory.
3. **Backend (Render/Railway)**: Point to the `server` directory.

> [!NOTE]
> See `deployment_guide.md` for step-by-step instructions with real-time sync setup.
